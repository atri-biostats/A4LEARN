# biomarker_AB_Test: Biomarker - Amyloid-β Tests

ATRI external analysis dataset.

## Usage

``` r
data(biomarker_AB_Test)
```

## Format

A data frame with 31480 rows and 12 variables.

## Details

- SUBSTUDY: NA

- BID: NA

- VISCODE: NA

- SAMPLE_ID: Sample Identifier

- LBNAM: Laboratory Name

- LBSPEC: Specimen Material Type

- LBCAT: Category for Lab Test

- LBTESTCD: Short Name of Lab Test

- LBORRES: Lab Result

- LBORRESU: Lab Results Units

- LBMETHOD: Method Code for Test 

- LBREASND: Reason Test Not Done
