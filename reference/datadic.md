# A4 LEARN data dictionaries

These data files contain meta data for other data files in this package.

## Usage

``` r
data(clinical_datadic)

data(derived_datadic)

data(external_datadic)

data(visits_datadic)

derived_datadic

external_datadic

visits_datadic
```

## Format

A data frame

An object of class `spec_tbl_df` (inherits from `tbl_df`, `tbl`,
`data.frame`) with 207 rows and 6 columns.

An object of class `spec_tbl_df` (inherits from `tbl_df`, `tbl`,
`data.frame`) with 835 rows and 6 columns.

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 152
rows and 6 columns.
