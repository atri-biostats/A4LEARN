# cogdigit: Cognitive Test - Digit Symbol Substitution Test

ATRI clinical analysis dataset.

## Usage

``` r
data(cogdigit)
```

## Format

A data frame with 26769 rows and 7 variables.

## Details

- SUBSTUDY: NA

- BID: NA

- VISCODE: NA

- DONE: Was assessment/procedure done?

- NDREASON: If No, reason not done:

- VERSION: Version

- DIGITTOTAL: Total Score
