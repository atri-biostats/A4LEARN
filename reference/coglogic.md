# coglogic: Cognitive Test - Logical Memory

ATRI clinical analysis dataset.

## Usage

``` r
data(coglogic)
```

## Format

A data frame with 26787 rows and 8 variables.

## Details

- SUBSTUDY: NA

- BID: NA

- VISCODE: NA

- DONE: Was assessment/procedure done?

- NDREASON: If No, reason not done:

- LMSTORY: Version

- LIMMTOTAL: Immediate Recall Total Score

- LDELTOTAL: Delayed Recall Total Score
