# habits: Habits

ATRI clinical analysis dataset.

## Usage

``` r
data(habits)
```

## Format

A data frame with 14306 rows and 13 variables.

## Details

- SUBSTUDY: NA

- BID: NA

- VISCODE: NA

- DONE: Was assessment/procedure done?

- NDREASON: If No, reason not done:

- SMOKE: Average number of packs smoked per day:

- ALCOHOL: Average number of alcoholic drinks consumed per day:

- CAFFEINE: Average number of cups of caffeine consumed per day:

- SUBUSE: Has the participant used other substances (e.g.
  non-pharmacologic substances such as medical marijuana)?

- AEROBIC: Average number of hours of aerobic exercise (i.e. jogging,
  swimming, bicycling) PER WEEK:

- WALKING: Average number of minutes of walking per day:

- SLEEP: Average total number of hours slept at night:

- SLEEPDAY: Average total number of minutes napped during the day:
