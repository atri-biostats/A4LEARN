# biomarker_Plasma_Roche_Results: Biomarker - Roche Plasma Assay

ATRI external analysis dataset.

## Usage

``` r
data(biomarker_Plasma_Roche_Results)
```

## Format

A data frame with 13418 rows and 14 variables.

## Details

- SUBSTUDY: NA

- BID: NA

- VISCODE: NA

- LABNAM: Laboratory Name

- LABD_DAYS_CONSENT: Date Specimen Collection (Derived to days since
  study consent date)

- LABD_DAYS_T0: Date Specimen Collection (Derived to days since study
  start date \[A4: Ranomization date \| LEARN: Baseline registry exam
  date\])

- SAMPLE_ID: Sample Identifier

- LBSPEC: Specimen Material Type

- LBMETHOD: Method Code for Test

- LBTESTCD: Short Name of Lab Test

- LBMTDL: Name of Lab Test

- LABRESN: Lab Result

- LABORESU: Lab Results Units

- LABRESC: Reason Not Done
